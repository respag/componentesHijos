Componente hijo que recibe información desde el Padre

Este es un ejemplo del uso de un componente hijo en Angular 7 que recibe del padre el nombre de un país y llena sus provincias y sus ciudades usando un archivo json

En el primer caso el componente hijo recibe la información cuando el usuario selecciona un país desde el DropDowList que lista todos los países diferentes leídos desde el archivo Json.

Después se usan 3 diferentes instancias del componente hijo, a los que se les pasan un país fijo: México, Venezuela e Italia, respectivamente



Como se ve, cada componente es independiente y muestra las provincias del país que se le pasa desde el componente padre, y una vez elegida la provincia, en su segundo DropDownList, muestra las ciudades de esa provincia.
